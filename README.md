# ls-auth-ui #

## Run with docker-compose (for local development) ##

* git clone the repo
* `docker-compose build`
* `docker-compose up`
* `docker-compose run web rake db:create db:migrate`

The ls-auth-ui app should then be accessible at localhost:3000.
